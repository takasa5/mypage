Orbit

2017/08~

目的

自分が、音楽は好きだが弾くことができないので、かんたんに音を作れればいいなと思って制作した。 もともとはTkinterを用いたデスクトップアプリケーションで作っていたが、見た目が気に入らなかったことと、 より多くの人に見てもらうためにはWebで作るのがよいと思い、Webアプリケーションとして作り直した。

言語

  • JavaScript(jQuery)
  • HTML

概要

画面上がプレイエリアで、画面下が操作用のボタン等。 プレイエリアに「オービット」や「プラネット」を置いていく。 再生ボタンを押すとオービットが回転し、プラネットと衝突することで音が鳴る。 オービットの速度や半径、位置を変えることで変わっていくリズムを楽しむ。

stardust

2017/10~

目的

星を見るのが好きでよく天体観測に行っていたが、星座の知識はなく、なかなか星座を探すのが難しかった。 実際に見ているときは、スマートフォンをかざせば大体その辺にある星座を表示してくれるアプリを使えばよかったが、 撮影した写真については使えない。そこで、写真を入力とし、星座を探してくれるプログラムを作ろうと思い立った。

言語

  • Python

概要

画像処理プログラムと独自の星座データからなる。画像処理は効率化・高速化に注力しているが、専ら問題となるのは星座データの少なさである。 特にディープラーニング等をしているわけではないので一つの星座につき数例あれば手作業でデータが作れるのだが、検索などでは意外とあまり見つからない。 そこで、広くプログラムを使用可能としデータの提供を期待する「みちびき」の開発へとつながる。
ディープラーニングの使用は検討中。

みちびき

2018/01~

目的

星座検出プログラムstardustを作ってみたのはいいものの、手元に星座の写真がなくて種類が増えない。 そこで、広くプログラムを使用可能としユーザからのデータの提供を期待するもの。
LINE botとWebアプリケーションの2種類がある。

言語

  • Python
  • JavaScript(jQuery)
  • HTML

概要

LINE botでもWebアプリケーションでも、手順をわかりやすくすることに努めた。 星空の画像はサイズが大きいことが多いので、サーバとの送受信に時間がかかる。 いかに素早く処理をするかということが課題になっている。 その他さまざまな課題(主に無料サーバを使っているせい)によりWebは未公開だが、LINE botは公開まで達成。

Dooune

2018/06~

目的

Orbit同様、かんたんな音作りができることを目指して作り始める。 様々なJavaScriptライブラリを使ってみたかったのも目的の一つ。

言語

  • JavaScript(jQuery)

概要

画面全体がプレイエリアで、タッチやドラッグで線が描ける。 円や四角形を描くと対応する図形が画面に出現し、物理シミュレーションで落ちたり跳ねたりする。 地面や図形同士でぶつかると、音が出る……という、「落書き+音が出る」な単純な仕組み。 Web上で多数の音を同時に再生すると音が割れる問題があり、なかなかうまくいかない。 そもそもOrbitでもそうだったが、簡単にきれいな音が出せるシステムを作るためには音楽の才能が必要だと思う。 現状、あんまり心地よくない音しか出せない。

概要

最近若者に流行っている「整う」というアレ。「サウナ」と便宜的に呼びはするもののその本質はサウナそのものではなく、 サウナ→水風呂→外気浴を自分に合った時間行うことによるトリップ状態(通称「整う」)。 お金がかからず、気持ちがよく、健康的(たぶん)なので、おすすめの趣味である。 なのに人を誘うと「暑いの苦手だから」「長く入れないから」と断られるのは不本意である。そうではないのだ。 このページでは、体験したサウナとそこへのコメントを記録したい。

サウナセンター大泉(鶯谷)

2018/03/10, 04/22
高砂サウナ歴原点にして頂点。クールダウンルームという冷蔵庫みたいな部屋がありこれが最高に整う。

萩の湯(鶯谷)

2018/04/22
湯の種類が豊富で良い。施設の大衆ウケするきれいさのわりにサウナ、水風呂、椅子の配置や給水ポイントがサウナー向けっぽい。

清水湯(武蔵小山)

2018/05/05
縦長な形でせまい。露天風呂の風通しが良くて気持ちよかった記憶。

湯けむりの庄(綱島)

2018/07/15
サウナがめっちゃ広くて高温。塩サウナもある(高砂は嫌い)。露天風呂スペースは広く、寝湯がとてもリラックスでき良い。

美肌の湯(埼玉県北本市)

2018/09/15
ここも寝湯があって良かった。2019年1月に閉店しちゃうらしい。

星音の湯(秩父)

2018/09/16
施設はりっぱだがサウナの温度が低くて残念。夜は星が見えるというウワサがある。

そしがや温泉21(祖師谷)

2018/09/23
プールがある。サウナ、ミストサウナ、冷凍サウナがあるがどれも狭く、整いに行くよりプールを楽しむ温泉。
